# Tourism & Hospitality Analytics Dashboard – Tableau

## Project Overview

This Tableau project analyzes tourism and hospitality data to uncover trends in tourist arrivals, hotel performance, revenue generation, tourist satisfaction, and economic impact. The dashboard provides interactive visualizations that help users explore tourism performance across different destinations and time periods.

## Objectives

* Analyze tourism trends and visitor patterns.
* Evaluate hotel occupancy rates and ratings.
* Monitor tourism revenue and economic impact.
* Identify top-performing destinations.
* Present insights through interactive visualizations.

## Tools Used

* Tableau
* Data Visualization
* Calculated Fields
* Filters and Parameters
* Dashboard Design

## Key Features

* Interactive dashboard with dynamic filters.
* Tourism performance analysis across destinations.
* Revenue and tourist satisfaction tracking.
* Hotel occupancy and rating analysis.
* Maps, charts, and KPI visualizations.

## Insights Generated

* Identified top tourism destinations.
* Analyzed tourism revenue trends.
* Evaluated hotel performance metrics.
* Explored factors affecting tourist satisfaction.
